# Merchant_Using_Hibernate
Merchant Product Management System is a Java-based application developed using Hibernate and MySQL to manage merchants and the products they sell. It implements a One-to-Many relationship in which a single merchant can have multiple products associated with them. The application supports complete CRUD operations to add, view, update, and delete both merchant and product records. This project demonstrates core Hibernate concepts such as entity mapping with annotations, session and transaction management, and seamless integration with a relational database.
